# 🌿 EcoFusion - Professional Sustainability Platform

A modern, clean, and professional fintech-style platform for recycling. Earn TrashCash (TCC) tokens and redeem real rewards.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit: **http://localhost:8081/**
